**AgentFlow** is a unified, extensible all-in-one open-source framework designed for synthesizing high-quality agent training and evaluation data.

### Key Features

- 🔁 **Unified synthesis environment** for agent trajectory, reasoning trace, tool interaction, and environment feedback
- 📦 **Scalable pipeline** supporting diverse task types and tool ecosystems
- 🧪 **High-quality data generation** for training and evaluating LLM-based agents

### My Contribution

I serve as a **core developer** of AgentFlow, contributing to the framework design, synthesis pipeline, and benchmark integration. This project underpins multiple research efforts including BrowseComp-V³ and MMDeepSearch.

### Affiliation

Developed at the **International Center for Machine Learning, Peking University**, under Prof. Wentao Zhang's research group.
