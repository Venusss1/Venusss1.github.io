**MMDeepSearch** is an ongoing research project at PKU ICML Center, aiming to improve multimodal LLMs' ability to answer questions through real-world web search via automated data synthesis and post-training.

### My Role

I am responsible for the **SFT/RL training architecture and strategy**, including:

- **Search-R1 Reconstruction**: Reconstructed the classic Search-R1 project using Verl's latest **AgentLoop + Function Calling** paradigm, extending it to multimodal retrieval scenarios
- **DeepEyes SFT**: Reproduced the SFT part of the DeepEyes multimodal retrieval framework
- **Training Infrastructure**: Extensive experience with 16× A100 / H20 GPU clusters using DeepSpeed, Verl, LlamaFactory, and ms-swift

### Baselines & Evaluation

Familiar with classical multimodal retrieval baselines: **BrowseComp-VL**, **FVQA**, **SimpleVQA**, and evaluation via **VLMEvalKit**.
